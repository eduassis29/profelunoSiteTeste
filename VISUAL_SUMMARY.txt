1️⃣  GERAR APP_KEY

    │                                                                     │
    │ → Copie a saída (será algo como: base64:xY3zAb...)               │
    └─────────────────────────────────────────────────────────────────────┘

2️⃣  CRIAR CONTA RAILWAY
    → Visite: https://railway.app
    → Login com GitHub
    → Clique "New Project"

3️⃣  CONECTAR GITHUB
    → Selecione "Deploy from GitHub"
    → Autorize Railway
    → Selecione seu repositório "profeluno"

4️⃣  CRIAR DATABASE
    → No Railway: "New Service"
    → Selecione: "Database > PostgreSQL"
    → Railway criará credenciais automaticamente

5️⃣  ADICIONAR VARIÁVEIS (Railway Dashboard)
    ┌─────────────────────────────────────────────────────────────────────┐
    │ APP_KEY              = [pasta aqui do passo 1]                     │
    │ APP_ENV              = production                                   │
    │ APP_DEBUG            = false                                        │
    │ DB_DATABASE          = profeluno                                    │
    │ DB_USERNAME          = postgres                                     │
    └─────────────────────────────────────────────────────────────────────┘

6️⃣  FAZER PUSH PARA GITHUB
    ┌─────────────────────────────────────────────────────────────────────┐
    │ git add .                                                           │
    │ git commit -m "Add Railway configuration"                         │
    │ git push origin main                                               │
    └─────────────────────────────────────────────────────────────────────┘

7️⃣  DEIXA RAILWAY FAZER A MÁGICA! 🚀
    → Railway detecta automaticamente
    → Build das imagens (2-3 min)
    → Deploy dos serviços
    → Sua app está LIVE!
